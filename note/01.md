## 🛠️ 第一阶段：搭建游戏世界 (Day 1)

**目标**：创建一个看起来舒服的 3D 场景，设置好相机角度。

### 1. 基础设置
*   **创建项目**：(你已经完成了)
*   **设置分辨率**：
    *   点击菜单栏 `项目 (Project)` -> `项目设置 (Project Settings)`。
    *   在 `Display` -> `Window` 中：
        *   `Viewport Width`: 720 (或其他竖屏比例宽度)
        *   `Viewport Height`: 1280
        *   `Mode`: Canvas Items (保持默认即可，或者选 Windowed)
        *   `Stretch` -> `Mode`: `canvas_items` (确保在网页上能缩放)
        *   `Stretch` -> `Aspect`: `keep` (保持长宽比)

### 2. 创建主场景 (Main Scene)
*   创建一个 **3D 场景** (根节点选 `Node3D`)，命名为 `Main`。
*   保存场景为 `res://scenes/main.tscn` (建议创建 `scenes` 文件夹)。

### 3. 设置环境 (Environment)
*   添加一个 `WorldEnvironment` 节点。
*   在右侧属性面板，`Environment` -> `New Environment`。
*   点击新建的 Environment，设置 `Background` -> `Mode` 为 `Sky` 或 `Color` (跳一跳通常是纯色背景，显得干净)。
*   添加一个 `DirectionalLight3D` (平行光)，开启 `Shadow` (阴影)，调整旋转角度，让物体有立体感。

### 4. 设置相机 (Camera)
这是跳一跳视觉风格的关键！
*   添加 `Camera3D` 节点。
*   **位置与角度**：
    *   将相机抬高 (Y轴增加)。
    *   将相机后退 (Z轴增加)。
    *   旋转相机向下看 (X轴旋转 -30 到 -45 度)。
    *   *提示*：跳一跳通常是 **等轴测视角 (Isometric)** 或 **正交视角**。
*   **投影模式 (Projection)**：
    *   将 `Projection` 属性改为 **`Orthogonal` (正交)**。这意味着物体不会因为远近而产生透视形变（近大远小），这对于判断距离非常重要。
    *   调整 `Size` 属性，直到画面中的物体大小合适。

### 5. 创建第一个平台 (Base Platform)
*   添加一个 `MeshInstance3D` 作为起始台子。
*   `Mesh`: 选择 `BoxMesh` (立方体) 或 `CylinderMesh` (圆柱体)。
*   `Material`: 给它一个简单的材质颜色。
*   **StaticBody3D**: 选中 Mesh，点击编辑器视口上方的 `Mesh` 菜单 -> `Create Trimesh Static Body` 或 `Create Single Convex Collision Sibling`。这将自动创建碰撞体，防止主角掉下去。

> **🎓 知识点小贴士**：
> *   **正交投影 (Orthogonal Projection)**：与人眼看到的“透视投影”不同，正交投影中平行的线永远平行。常用于策略游戏或像跳一跳这样的精确距离判断游戏。
> *   **StaticBody3D**：静态刚体。用于地板、墙壁等不会动的物体。它会阻挡玩家，但不会被推开。
